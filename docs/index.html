What's included

Zod is a modular AI agent platform built for developers who want to build AI-powered applications. Designed for flexibility and control, with everything you need to create powerful AI assistants.

From simple chatbots to complex autonomous agents, Zod provides the building blocks. Mix and match capabilities to create exactly what you need — no bloat, no compromises.

Whether you're building a customer support system, a coding assistant, or a full automation platform, Zod scales with your needs.

Built by experts — designed by developers who understand AI systems inside and out, with a focus on reliability and safety.

Extensible — add new capabilities through extensions, tools, and skills that integrate seamlessly with the core engine.

Safety first — built-in policy enforcement, budget limits, step limits, and approval gating keep your AI controlled.

Real-time streaming — see what your AI is thinking as it thinks it, with full event logging and debugging.

Multi-provider — connect to OpenAI, Anthropic, Google, and any OpenAI-compatible API.

Open source — free to use, modify, and extend. Built in public, for the community.

Frequently asked questions

Everything you need to know.

General

Is Zod really free to use?

Yes! Zod is completely open source under the MIT license. You can use it for personal or commercial projects without any restrictions.

What AI providers are supported?

Zod supports OpenAI, Anthropic, Google, and any OpenAI-compatible API. Adding new providers is straightforward through the extension system.

Technical

What technologies does Zod use?

The core harness is built with TypeScript and Node.js. The client interface uses SolidJS for a fast, reactive browser experience.

Can I extend Zod with custom tools?

Absolutely! The extension system is designed for this. You can add custom tools, skills, and even entirely new AI providers with minimal effort.

How does the safety system work?

Zod includes built-in policy enforcement, budget limits, step limits, and loop detection. You can also configure approval gating for sensitive operations.

Licensing

Can I use Zod for client projects?

Yes! The MIT license allows you to use Zod in any project, including client work and commercial products.

Where can I get help or support?

Join our GitHub Discussions or Discord community for help. You can also file issues on GitHub for bugs or feature requests.
